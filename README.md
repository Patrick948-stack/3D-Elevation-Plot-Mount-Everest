# ⛰️ 3D Elevation Plot – Mount Everest

An interactive 3D terrain visualization of Mount Everest built with JavaScript and Plotly.js.  
This project loads a real-world elevation dataset (CSV format), processes it in the browser, and renders a fully interactive 3D surface.

---

## ⚠️ Important: Elevation Dataset Not Included

The dataset used in this project — **`everest_elevations.csv`** — is **too large to be hosted on GitHub** (GitHub’s file-size limit is 100 MB).  
Attempts to compress the file were unsuccessful, so the dataset is **not included in this repository**.
